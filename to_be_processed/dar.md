To give.

Example: 'Dame las llaves'.

Dar conjugation includes irregularities in most forms of conjugation.

# Indicative Present

| tense    | conjugation |
| -------- | ----------- |
| yo       | doy         |
| tú       | das         |
| él       | da          |
| nosotros | damos       |
| ellos    | dan         |

# Indicative Preterite

| tense | conjugation |
| ----- | ----------- |
| yo    | dí          |
| tú    | diste       |
| el    | dio         |
| nos   | dimos       |
| ellos | dieron      |

So it conjugates as though it is dir or dor.
